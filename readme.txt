SkyHope智能解题系统
环境：
任意Web服务器(如Apache，Nginx)，需开启Https！SSL证书请自行获取！
Html+Css+Js支持
PHP8.0+
Mysql8.0+
需联网从CDN获取部分Css样式图标等
Python3.0以上(运行TxYuanbao-To-PyAPI)
运行方法：
1.确保配置好环境，Web服务器指定SkyHopeZyb为根目录
2.运行/SkyHopeZyb/TxYuanbao-To-PyAPI内程序，具体方法见https://github.com/TW-SkyHope/TxYuanbao-To-PyAPI
3.确保TxYuanbao-To-PyAPI正常运行(默认8000端口，若有需要请更改tack.php中127.0.0.1:8000与aiapi.py最下方端口)，访问/index.php文件，进行安装
4.所有配置完成后即可正常使用
